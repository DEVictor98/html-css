Tipos de Formatação

Negrito/destaque

para usar negrito com a semantica correta utilizasse a tag STRONG para deixar assim

Marcação de texto

para usar a marcação de texto usamos a tag MARK para que fique assim

Textos grandes e pequenos

não utilizamos mais a tag BIG pois é obsoleta, entretanto para diminuir usamos a tag SMALL exemplo big exemplo small

Textos excluidos

para textos excluidos se faz uso da tag DEL exemplo

Sublinhar

para sublinhar e ainda ficar dentro das novas regras do HTML5 se faz uso da tag INS exemplo

Sobre e sub escrever

para sobrescrever se faz uso da tag SUP exemplo: X20-45
e para Subescrever algo utilizasse da tag SUB exemplo: H2O
